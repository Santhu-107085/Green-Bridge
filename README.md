# Green-Bridge
GreenBridge is a full-stack web application designed to connect rice farmers directly with buyers such as rice mill owners and retailers. The platform focuses on eliminating middlemen, improving farmer profits, and ensuring fair pricing through a simple, user-friendly interface.
It provides features like user registration, role-based dashboards (Buyer/Seller), rice listing, price tracking, AI-based insights, and geolocation-based discovery of nearby buyers and sellers. The system is designed to work efficiently even in low-resource environments, making it accessible to small-scale farmers.
##How to Run::
git clone https://github.com/your-username/greenbridge.git
cd greenbridge
python -m venv venv
